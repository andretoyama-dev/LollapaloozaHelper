# Lollapalooza Helper

A modern group-oriented web dashboard designed to organize music festival schedules and coordinate concert attendance among groups of friends. 

## Features

- **Bilingual Interface (EN/PT)**: Complete translation coverage between English and Brazilian Portuguese, enabling quick language switching.
- **Group Scheduling**: Select favorite bands, view friends' choices, and identify who is going where in real-time.
- **Visual Schedule Conflict Warnings**: Automated notifications and visual indicators when overlapping time slots are selected.
- **Group Timeline Visualization**: Visual comparison of friend agendas throughout the day from 12:00 to 00:00.
- **Express Backend Middleware**: Serves static pages dynamically and provides basic health metrics.
- **Real-time Synchronization**: Powered by Supabase Realtime to update selections instantly across all connected screens.

## Tech Stack & Tools

- **Frontend**: React 19, TypeScript, Tailwind CSS, Motion (framer-motion)
- **Database & Realtime**: Supabase
- **Hosting**: Vercel
- **Build Tool**: Vite 6
- **Server Middleware**: Express
- **Icons**: Lucide React
- **AI Tools**: Google AI Studio (initial generation), Antigravity + Claude (refinement and iteration)
